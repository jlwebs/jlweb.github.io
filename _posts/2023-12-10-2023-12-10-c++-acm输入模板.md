5.while暴读逻辑
如果输入中包含回车、制表符（Tab）或其他转义符，cin >> n 默认情况下会跳过它们并继续读取下一个有效的输入。对于输入中包含的空格，cin >> n 也会将它们视为分隔符，因此会忽略它们。
cin>>str 字符串只能接收一个单词

7.完整案例
```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
    int n;
    int nn;

    while(cin >> n){
        string s;
        std::vector<int> v; 
        getchar();
        while(n--){
            cin >> nn ;
            getchar();
            getline(cin, s);
            stringstream ss(s);
            string each;
            while(getline(ss, each, ' ')){
                v.push_back(stoi(each));
            }
            cout<<accumulate(v.begin(), v.end(), 0);
            if(n!=0) cout<<endl<<endl;
            else cout<<endl;
            v.clear();
        }
    }
    return 0;
}


```
